# 🎓 AI Student Tutor

**Name**:Emaan Noor  
**Roll No**: 2k24/AI/21 

## About Project
An AI-powered web app that generates lessons, explanations, 
and quizzes on any topic for students instantly!

## 🌐 Live Demo
https://huggingface.co/spaces/Anthophile/ai-student-tutor

## ✨ Features
- Enter any topic and get instant explanation
- 3 difficulty levels: Beginner, Intermediate, Advanced
- Generate Full Lesson, Explanation Only, or Quiz
- Powered by Llama 3 AI model

## 🛠️ Tech Stack
- Python
- Gradio
- Groq API (Llama 3)
- HuggingFace Spaces (Deployment)

## 📁 Files
- app.py — Main application code
- requirements.txt — Dependencies
